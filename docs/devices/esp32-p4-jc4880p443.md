# Guition ESP32-P4 JC4880P443 (4.3")

4.3" (480×800) portrait touchscreen panel with ESP32-P4 and ESP32-C6 coprocessor for WiFi/Bluetooth. Uses a portrait layout: album art at the top, track info below on an accent-colour panel, and play/pause in the bottom-right.

## Install

Click **Install** to flash the firmware directly from your browser. No ESPHome dashboard or YAML required.

<InstallButton device="jc4880p443" />

::: tip
If you prefer to install via the ESPHome dashboard, see [ESPHome Config](/advanced/esphome-config) (this device uses the `devices/guition-esp32-p4-jc4880p443` package).
:::

## Notes

This device should be tested on real hardware after flashing. If touch input is mirrored or rotated, adjust only the GT911 touch transform in the device package.
